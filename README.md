# FunnyJsonExplorer
Funny JSON Explorer（FJE），是一个JSON文件可视化的命令行界面小工具
V1版本为初次尝试，没用组合模式，V2是在V1基础上增加组合模式，V3是在V2基础上进一步理解使用组合模式。FunnyJsonExplorer2是在V3基础上改用者迭代器+策略模式。

SYSU 2024软件工程 作业
